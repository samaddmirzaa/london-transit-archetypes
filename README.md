# London Transit Archetypes

Unsupervised discovery of behavioural archetypes among London's 471 rail
stations, using TfL NUMBAT 2024 quarter-hourly demand data across five
daytypes.

**Status:** in development.

## Data source
TfL NUMBAT 2024 outputs, published October 2025.
Available from: https://crowding.data.tfl.gov.uk/

## Setup
```bash
conda env create -f environment.yml
conda activate london-transit
```

More documentation will follow as the project develops.
